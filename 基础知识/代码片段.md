### 统计时间  

```python

```


### 时间格式化  


### 数字格式化输出的标准写法  

```python 
In [19]: x = 1.23456

In [20]: print(f'{x:.2f}')
1.23

In [21]: print(f'{x:.3f}')
1.235
```

### 转化成百分数  

```python 
In [1]: num = 0.8765

In [2]: print(f'{num * 100:.2f}%')
87.65%
```
