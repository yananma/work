

### read_json    

```python 
def read_json(file_name):  
    """读取 JSON 文件"""  
    with open(file_name, 'r') as f:  
        return (line.strip() for line in f)  
```



### 合并 json  

```python 
def concate_json(old_file, new_file):
    """合并 JSON 文件"""
    with open(old_file, 'a', encoding='utf-8') as f:
        with open(new_file, 'r', encoding='utf-8') as nf:
            for line in nf:
                f.write(line)
```
